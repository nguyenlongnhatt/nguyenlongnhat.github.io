GIỚI THIỆU CHUNG

Khóa học Bảo mật Web là một khóa học cực kì hữu ích cho tất cả các lập trình viên trong quá trình xây dựng, vận hành web nhằm hạn chế tối đa khả năng bị hacker tấn công, mất dữ liệu…

MỤC TIÊU KHÓA HỌC

Giúp học viên nắm được :

  1. Nguyên nhân và cách phòng tránh các lỗ hổng bảo mật trong quá trình phát triển ứng dụng web.
  2. Sử dụng các công cụ để rà quét, phát hiện các lỗ hổng trong ứng dụng.

ĐỐI TƯỢNG HỌC VIÊN

Lập trình viên/ Sinh viên CNTT đã nắm vững Front end và Back end yêu thích lĩnh vực Bảo mật

YÊU CẦU HỌC VIÊN

  1. Thái độ học tập nghiêm túc
  2. Cam kết KHÔNG sử dụng kiến thức vào mục đích xấu

CẤU TRÚC BUỔI HỌC

Khóa học bao gồm 12 buổi * 3 tiếng, 1 tuần 3 buổi

Mỗi module bài giảng bao gồm:

  1. Giới thiệu lỗ hổng
  2. Nguyên nhân: Trình bày ví dụ code chứa lỗ hổng
  3. Cách khắc phục: Trình bày ví dụ code an toàn trên các ngôn ngữ như Java, C#, PHP
  4. Cách khai thác lỗ hổng, học viên thực hành trên lab từ dễ đến khó
  5. Công cụ hỗ trợ
  6. Bài tập về nhà

GIÁO TRÌNH

  • Hệ thống giáo trình bao gồm : Slide, video, code mẫu chi tiết giúp bạn chủ động tìm hiểu bài trước và ôn tập các kiến thức.
  • Hệ thống học LMS giúp bạn truy cập giáo trình mọi lúc mọi nơi, bạn chỉ cần đăng nhập là có thể học tiếp. Quyền sở hữu giáo trình vĩnh viễn.

NỘI DUNG CHƯƠNG TRÌNH

  1. Giới thiệu tổng quan khóa học, các khái niệm, công cụ, công nghệ web phổ biến. Lắng nghe bài toán bảo mật ứng dụng web của doanh nghiệp
  2. Injection(SQL, NoSQL Injection, OS Command, SSTI)
  3. Broken Authentication
  4. Sensitive Data Exposure
  5. XML External Entities (XXE)
  6. Using Components with Known Vulnerabilities, Insufficient Logging & Monitoring
  7. Broken Access Control
  8. Security Misconfiguration
  9. Cross-Site Scripting (XSS)
  10. Insecure Deserialization
